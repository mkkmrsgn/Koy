Usage of the Playfair Cipher¶

The Playfair Cipher is a digraph substitution cipher that encrypts pairs of letters. This document outlines the required inputs, processing steps, and the rules for encryption and decryption.

Input Requirements¶

  • Words must not contain the letter "J".
  • Enter a keyword.
  • Enter a message to encrypt or decrypt.

Processing Steps¶

  1. Remove the letter J from both the keyword and the message.
  2. Insert the desired keyword into the 5x5 table, ensuring no duplicates.
  3. Fill the table with the remaining letters of the alphabet (excluding "J").
  4. Split your message into pairs: - If two identical letters are found in a pair, separate them with an "X". - If there is an odd letter at the end of the message, append an "X".
  5. Ignore all spaces in the message.

Encryption and Decryption Rules¶

  • Rule 1: Move each letter down ONE. If it reaches the end, wrap around to the top.
  • Rule 2: Move each letter right ONE. If it reaches the end, wrap around to the beginning.
  • Rule 3: Swap letters with the ones on the opposite ends of the rectangle formed by the pairs.
